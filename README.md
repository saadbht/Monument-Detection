# Monument Detection (YOLO)

Project by **Saad Tozibar Rahman**.

A YOLO based monument detection pipeline with a small Flask web UI. 

## Quick Start
- Install: `pip install -r requirements.txt`
- Run: `python application.py`
- Upload an image and view the preview in the browser.

## Included
- `best.pt` (pretrained weights for the demo)
- `output/output.jpg` and `output/color_img.jpg` (example output)
- `application.py` + `detector_test.py` (web app + inference)

Dataset reference: https://universe.roboflow.com/kgitbank/landmark-detection-jqu66
